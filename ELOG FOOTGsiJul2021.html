  14   Wed Jul 14 15:45:36 2021 A. SartiInfoReconstructionData access

As data are starting to come, here are some infos on how to access it.

a) Data will be collected, at GSI, under

/lustre/bio/foot/GSI2021/

Inside this folder you will also find the TCALIB folder that contains the time calibration for EACH run collected.

b) To access the data at GSI under /lustre you need an account on the Virgo cluster.

Infos about Virgo:

Virgo User Manual:  https://hpc.gsi.de/virgo

https://hpc.gsi.de/virgo/slurm/environment.html

https://hpc.gsi.de/virgo/platform/software.html

A shortlist for who has already an account at GSI on the Kronos cluster:

  • Since we are part of the 'bio' group that does not have a coordinator under slurm, the requests to have access to the Virgo cluster have to be sent to cluster-service@gsi.de.

  • Sign on to the HPC-INFO mailing list.

To login you need to setup a tunnel:

ssh -J lxpool virgo-centos7.hpc.gsi.de

c) To process the data interactively on a local installation you can:

../../bin/DecodeRaw -in  /path/to/your/favourite.data.moved -out YourFavouriteName_Out.root -exp GSI2021 -run XXXX  

Please remember that XXXX must be a valid file name, specified in the campaign file, and that is linked to a valid, existing calib/GSI2021/WDTimeCalibration/tcalib_XXXX.dat file.
